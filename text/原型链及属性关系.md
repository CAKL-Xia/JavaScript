## prototype

在javascript中所有函数都有 prototype 属性。如下例子中

```javascript
funtion Person(){ 

}
Person.prototype.name = 'inherit'
const person1 = new Person();
const person2 = new Person();
console.log(person1.name) // inherit
console.log(person2.name) // inherit
```
